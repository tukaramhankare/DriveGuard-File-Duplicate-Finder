What it does

  • SHA-256 integrity scan

    Point it at a drive, folder, or media device — every file gets hashed in one pass.

  • Duplicates & corruption

    Groups identical files, flags unreadable/empty files, signature-vs-extension mismatches, and files whose hash changed since the last scan.

  • Export / Import SHA-256

    Save a scan to CSV and re-verify it later against the same files, reusing the same hashing and grouping logic.

  • Act at the original location

    Right-click or long-press: reveal, rename, copy/backup (hash-verified, collision-safe), replace (verified, auto-backed-up), or delete to a recoverable quarantine folder.

  • Full operation log

    Every action recorded, with a one-click Clear Log that only touches the log — never your files.

  • Dark & Light themes

    Toggle at runtime.

Build from source

Zero external Python dependencies — standard library only.

git clone https://github.com/tukaramhankare/DriveGuard-File-Duplicate-Finder.git
cd DriveGuard-File-Duplicate-Finder
python DriveGuard.py

To build your own .exe:

pyinstaller --onefile --noconsole --clean --hidden-import=tkinter --uac-admin --name DriveGuard DriveGuard.py
